# ML_01_Product_Classifier

This project builds a product catalog auto-classification workflow using product text such as titles and descriptions.

## Included in this repository

- Notebooks for preparation, cleaning, EDA, and modeling
- CRISP-DM planning and evaluation documents
- Small evaluation outputs and figures

## Excluded from GitHub

The following are intentionally ignored from version control:

- `data/` because the raw and prepared datasets are large and can be regenerated
- `models/` because trained artifacts are large generated outputs
- `download_amazon_large.py` because you chose not to publish the dataset download helper

## Notes

- GitHub has a hard size limit for normal files, so the current large data and model files should not be committed.
- If you later want to share trained models or datasets, prefer cloud storage, Hugging Face, Kaggle, or Git LFS.
